## Вступление
Полноценный исходник старого чита OneShot созданного под игру BlackRussia (игра создана на исходном коде reVC)

Это *не* готовый к использованию чит, некоторые функции устарели и чит нуждается в обновлении под последнюю версию игры, множество проблем. Поэтому настоятельно рекомендуется не использовать его, если у вас нет предварительных знаний по программированию или нет знаний по взлому игр в целом.

## Известные проблемы
- Краши.
- Нуждается в обновлении под актуальную версию игры.
- Только X32 бит.
